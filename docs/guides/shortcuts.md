# Keyboard Shortcuts Reference

Complete reference of all keyboard shortcuts available in Authorly for efficient content editing.

## Overview

Authorly provides comprehensive keyboard shortcuts for:
- Text formatting
- Block transformations
- Navigation
- Editing operations
- Undo/redo
- Selection

**Platform Note:** Use `Ctrl` on Windows/Linux and `Cmd` (⌘) on macOS.

## Text Formatting

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+B` | **Bold** | Make selected text bold |
| `Ctrl+I` | *Italic* | Make selected text italic |
| `Ctrl+U` | <u>Underline</u> | Underline selected text |
| `Ctrl+E` | `Inline Code` | Format selected text as inline code |
| `Ctrl+K` | Add Link | Insert or edit a link |

## Markdown Shortcuts

Authorly supports Markdown-style shortcuts:

### Headings

```
# Space → Heading 1
## Space → Heading 2
### Space → Heading 3
#### Space → Heading 4
##### Space → Heading 5
###### Space → Heading 6
```

### Lists

```
- Space → Bullet list
* Space → Bullet list
+ Space → Bullet list
1. Space → Numbered list
[] Space → Checklist (unchecked)
[x] Space → Checklist (checked)
```

### Other Blocks

```
> Space → Blockquote
``` Space → Code block
--- → Horizontal divider
*** → Horizontal divider
___ → Horizontal divider
```

### Inline Formatting

```
**text** → Bold
*text* → Italic
__text__ → Underline
~~text~~ → Strikethrough
`text` → Inline code
[text](url) → Link
```

## Special Behaviors

### Smart Enter

- **Empty Paragraph:** Creates new paragraph below
- **End of Heading:** Creates new paragraph below
- **End of List Item:** Creates new list item
- **Empty List Item:** Exits list (converts to paragraph)
- **Code Block:** Inserts newline within code

### Smart Backspace

- **Empty Block:** Deletes block and focuses previous
- **Start of Block:** Merges with previous block
- **Within Text:** Deletes character before cursor

### Smart Delete

- **Empty Block:** Deletes block and focuses next
- **End of Block:** Merges with next block
- **Within Text:** Deletes character after cursor

### Auto-Pairing

Characters are automatically paired:

| Type | Result |
|------|--------|
| `(` | `()` with cursor in middle |
| `{` | `{}` with cursor in middle |
| `[` | `[]` with cursor in middle |
| `"` | `""` with cursor in middle |
| `'` | `''` with cursor in middle |
| `` ` `` | ``` `` ``` with cursor in middle |

## Customizing Shortcuts

Currently, keyboard shortcuts are not customizable. If you need custom shortcuts, consider:

1. **Wrapper Component:** Intercept keydown events and call editor methods
2. **Custom Toolbar:** Create custom buttons with your preferred shortcuts
3. **Feature Request:** Request custom shortcut support on GitHub

### Example: Custom Shortcut Wrapper

```tsx
import { AuthorlyEditor } from 'authorly-editor';
import { useRef } from 'react';

function EditorWithCustomShortcuts() {
  const editorRef = useRef<any>(null);

  function handleKeyDown(e: React.KeyboardEvent) {
    // Custom: Ctrl+Shift+D for duplicate
    if (e.ctrlKey && e.shiftKey && e.key === 'D') {
      e.preventDefault();
      // Custom duplicate logic
      console.log('Custom duplicate');
    }
  }

  return (
    <div onKeyDown={handleKeyDown}>
      <AuthorlyEditor ref={editorRef} />
    </div>
  );
}
```

## Platform Differences

### Windows/Linux

- Uses `Ctrl` for all shortcuts
- `Ctrl+Y` for redo
- `Delete` key for forward delete

### macOS

- Uses `Cmd` (⌘) instead of `Ctrl`
- `Cmd+Shift+Z` for redo
- `Fn+Delete` for forward delete (on keyboards without dedicated Delete key)

## Tips for Efficient Editing

### 1. Learn Markdown Shortcuts

Type naturally using Markdown syntax - it's faster than using the toolbar.

### 2. Use Slash Commands

Press `/` to quickly insert any block type without reaching for the mouse.

### 3. Master Two-Level Selection

- First `Ctrl+A`: Select current block content
- Second `Ctrl+A`: Select all blocks in editor

### 4. Navigate with Keyboard

Use `Ctrl+↑/↓` to jump between blocks instead of clicking.

### 5. Duplicate Frequently Used Blocks

Press `Ctrl+D` to duplicate a block and modify it.
